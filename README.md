This repository hosts our team's **internal planning notes** for the IUT 12th ICT FEST Agentic AI Hackathon.

## On 24 July (Hackathon Day) — Submission Workflow

Organizers will share a **template repository** (or instruct teams to create a new repo). The flow will be:

1. Open the organizer-provided template repo URL
2. Click **"Use this template"** (or fork) → name it `agentic-ai-<your-team-name>`
3. Make it **public**
4. Clone your new repo locally
5. Build the entire project from scratch during the 24-hour window
6. Push commits throughout the day
7. At 10 AM on 25 July, submit your new repo URL in the Google Form

**The submission is judged on:**
- GitHub repo creation timestamp (must be ≥ 24 July 10 AM)
- First commit timestamp (must be ≥ 24 July 10 AM)
- Public visibility
- Quality of code, demo, and README

This `AI_agent_hackathon` repo stays as **internal planning only** and is NOT submitted.
